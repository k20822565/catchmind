# 캐치마인드 프로젝트 계획

## 프로젝트 컨셉
- 직장인 3~4명이 업무시간에 몰래 즐길 수 있는 가벼운 캐치마인드 게임
- 요란하지 않은 UI, 빠른 접속, 로그인 없음
- 데스크탑 전용

---

## 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | HTML5, CSS3, Vanilla JS |
| 실시간 통신 | Socket.io |
| 백엔드 | Node.js + Express |
| 배포 | Render.com (무료 플랜) |

---

## 기능 명세

### 1. 로비 / 방 입장
- 닉네임 입력 (최대 8자)
- 방 만들기: 랜덤 4자리 코드 생성 (예: `AB3K`)
- 방 입장: 코드 입력으로 참여
- 방 인원: 최대 6명 (최소 2명부터 게임 시작 가능)

### 2. 대기실
- 참여한 플레이어 목록 표시
- 방장이 라운드/시간 설정
  - 라운드: 1 / 2 / 3 / 4 / 5
  - 제한시간: 1분 / 2분 / 3분
- 방장만 게임 시작 버튼 활성화
- 방 코드 복사 버튼

### 3. 게임 진행
- 턴마다 순서대로 그리는 사람 지정 (방 입장 순)
- 그리는 사람에게만 단어 표시 (나머지는 `???` 표시)
- 단어는 WORDS 목록에서 랜덤 1개 강제 지정
- 제한시간 카운트다운 표시
- 정답 맞추면 즉시 공개 + 점수 부여

### 4. 그림판 (캔버스)
- 마우스 드래그로 자유 드로잉
- 색상 팔레트 (기본 8가지 색)
- 굵기 선택 (얇게 / 보통 / 굵게)
- 실행취소 (Ctrl+Z 또는 버튼)
- 전체 지우기 버튼
- 그리는 사람만 조작 가능, 나머지는 관전

### 5. 채팅
- **정답 입력창**: 별도 입력창, 제출 시 정답 판별
- **일반 채팅창**: 자유 대화 (힌트 요청, 잡담 등)
- 정답 맞히면 채팅창에 `[닉네임]님 정답!` 표시

### 6. 점수 / 결과
- 정답자: +점수 (남은 시간 비례)
- 그린 사람: 정답자가 있으면 +보너스
- 라운드 종료 후 현재 점수판 표시
- 전체 라운드 종료 후 최종 결과 화면

## UI 스타일 옵션

### Option A — "엑셀 위장형"
- 배경: 흰색, 테두리 전부 회색 1px 실선
- 폰트: `Malgun Gothic` (윈도우 기본), 작은 크기
- 헤더: 연한 회색 (`#f2f2f2`) 줄무늬
- 버튼: 회색 테두리, 회색 배경 — 마치 엑셀 툴바 버튼
- 색상 팔레트: 작은 사각형 셀처럼 배열
- 전체적으로 MS Excel 2010 느낌
- **장점**: 옆에서 봐도 일하는 것처럼 보임

### Option B — "심플 내부 툴형"
- 배경: `#f8f9fa` (연한 회색)
- 포인트 컬러: 남색 (`#3b5bdb`) 1가지만
- 폰트: `Pretendard` 또는 시스템 폰트
- 버튼: 둥근 모서리, 색은 최소화
- 카드/패널 구분이 명확하지만 화려하지 않음
- **장점**: 깔끔하고 현대적, 어떤 모니터에서도 무난

## 배포 계획 (Render.com)

1. GitHub 레포 생성 후 푸시
2. Render.com → New Web Service 연결
3. Build Command: `npm install`
4. Start Command: `node server/index.js`
5. 무료 플랜 사용 (슬립 모드 있음 — 첫 접속 시 30초 대기)
